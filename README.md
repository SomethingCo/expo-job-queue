# expo-job-queue

An SQLite based job queue for Expo

## Installation

```sh
npm install expo-job-queue
```

## Usage

```js
import ExpoJobQueue from "expo-job-queue";

// ...

const result = await ExpoJobQueue.multiply(3, 7);
```
